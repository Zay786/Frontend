# 🚀 Logistics Quotation System

## 📌 Overview
This is a full-stack logistics quotation system that generates price estimates and downloadable PDFs based on shipment details.

The system is built with:
- **Frontend:** React
- **Backend:** Node.js (Express)
- **ML API:** FastAPI (Python)
- **Database:** PostgreSQL

---

## 🧠 Features

- 📦 Generate logistics quotations
- 💰 Dynamic pricing logic (based on weight, route, service, commodity)
- 📄 Automatic PDF quotation generation
- 🗄️ Store quotations in PostgreSQL
- 🔗 Full API architecture (React → Node → Python)
